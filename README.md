# Dungeon Crawler
A 2d dungeon C++ game where the mission is to take the treasure and avoid being caught by monsters to escape the dungeon!

Collect the magical amulet to change the size of the map and take it to your advantage! 

## Running the program

To run the program, pull the repository into Linux Environment:

    1. Open a terminal
    2. Connect to Ubuntu Linux or use Linux 
    2. `g++ -o dungeon dungeoncrawler.cpp helper.cpp logic.cpp` to compile the program
    3. `./dungeon` to start the program
    4. `<difficulty> <level>` to choose the difficulty and the level of the map
    5. Enjoy!

Be safe adventurer!
